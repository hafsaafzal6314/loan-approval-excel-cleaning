# Loan Approval Data Cleaning using Excel

## Project Overview

This project focuses on cleaning a Loan Approval dataset downloaded from Kaggle using Microsoft Excel. The goal was to improve the quality of the dataset by identifying and handling missing values, making it suitable for further analysis and preprocessing.

---

## Dataset

* **Source:** Kaggle
* **Dataset:** Loan Approval Dataset
* **Tool Used:** Microsoft Excel

---

## Work Performed

* Imported the loan dataset into Microsoft Excel.
* Inspected the dataset to identify missing values.
* Used Excel filters to locate incomplete records.
* Handled missing values appropriately to improve data quality.
* Reviewed the dataset for consistency after cleaning.
* Saved the cleaned dataset for future analysis and machine learning tasks.

---

## Excel Features Used

* Filters
* Sort & Filter
* Missing Value Identification
* Basic Data Cleaning Techniques
